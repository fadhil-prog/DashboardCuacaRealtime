# DashboardCuacaRealtime
Fadhil Akbar(2315061037)

Sistem ini merupakan sistem yang menampilkan secara real time cuaca yang ada di berbagai kota. Di dalam sistem ini, terdapat beberapa tampilan yang sangat membantu dalam memberikan informasi cuaca, dimulai dari ramalan cuaca hari ini, dan 5 hari kedepan. Terdapat juga kecepatan angin, temperature, Kelembapan. Fitur di sistem ini terdiri dari fitur Pencarian, pergantian suhu dari Celcius ke Fahrenheit, beberapa setingan favorite kota, dan fitur mode gelap dan mode terang. Sistem ini dibuat secara realtime dengan API yang tersambung ke situs OpenWeatherMap.

API pada sistem ini dilakukan dengan cara mengunjungi situs https://openweathermap.org/. Kemudian mendaftar akun dengan melalui akun google. Setelah mendaftar akun, akan diberikan API_KEY yang akan dimasukkan ke dalam kode di HTML nya, jadi sistem ini harus terhubung ke internet untuk mengakses web tersebut, jika tidak terhubung ke internet, maka sistem tidak akan mendapatkan informasi cuacanya.
